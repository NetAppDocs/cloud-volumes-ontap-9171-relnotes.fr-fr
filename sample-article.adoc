= Le titre de la page s'affiche ici
:allow-uri-read: 
:icons: font
:imagesdir: ./media/


[role="lead"]
Utilisez ce modèle pour les articles, sujets, ou tout autre nom que vous leur donnez. Le fichier index.adoc ne contient pas les bons mots-clés pour une page d’article et l’utiliser comme modèle peut causer des problèmes de publication par la suite.


NOTE: Supprimez ce fichier avant la publication initiale, car il sera publié même s'il n'est pas lié à la barre latérale. Après avoir créé quelques articles, vous pouvez les utiliser comme point de départ à la place.

Si vos articles se trouvent dans un sous-dossier de contenu, vous devrez apporter deux modifications à la notation indiquée ci-dessus :

* `:imagesdir:../media/`
* `permalink: folder-name/sample-article.html`
